# 🏇 KEIBA DOMINATOR  
## ― データで勝率を制圧せよ ―

---

## 概要

KEIBA DOMINATOR は、  
JRA開催構造・月別ローテーション・グレードレース情報を解析し、  
その日のメインレースに最適化された抽選ロジックを提供する  
競馬意思決定支援エンジンです。

開催地、日付、グレード情報をもとに  
“今この瞬間のメインレース”を特定し、  
勝利番号を演算します。

---

## 主な機能

### 📅 月別開催場自動判定
- 現在月から開催場を抽出
- ローテーションを考慮した構造

---

### 📡 ICSデータ解析
- JRAレースカレンダー取得
- 今月開催のグレードレース抽出
- GI / GII / GIII 自動判定
- 開催日までの残日数算出

---

### 🎯 勝利番号決定ロジック


## 設計思想

- 表示サイズはCSS基準で統一
- グレード誤判定対策済み
- 年跨ぎ対応可能なICS取得設計
- 状態管理を明確化
- デバッグコード分離可能設計

見た目は派手でも、内部は堅実。

---

## 技術構成

- Vanilla JavaScript
- DOM制御
- ICSフォーマット解析
- 正規表現ベースのグレード抽出
- 月別開催マッピング

フレームワーク非依存設計。

---

## 使い方

1. 開催地を選択  
2. レースを選択  
3. 「運命を回す」を押す  
4. 表示された番号を受け入れる
