# 🏇 ULTIMATE KEIBA FINAL  
― 週末開催を制圧せよ ―  

Live Demo:  
https://doraiso.github.io/ultimate-keiba-final/

---

## 概要

ULTIMATE KEIBA FINAL は、  
JRA開催構造と公式カレンダーデータを解析し、  
**今週末に開催される競馬場とメインレース情報** を即座に提示する  
軽量・自動更新型の競馬支援ツールです。

開催地・日付・グレード情報をもとに  
“今この瞬間の開催構造” を可視化します。

---

## 主な機能

### 📅 週末開催地の自動判定
- JRA公式カレンダーJSONを解析
- ローカル開催（小倉・中京など）対応
- 週末開催構造を正確に抽出

### 🏆 重賞レース検出
- 該当日のグレードレースを自動判定
- GI / GII / GIII 判別
- 未来の重賞誤表示を防止

### 🎯 レース表示エンジン
- 当日の開催構造に基づきレース一覧を生成
- 重賞開催時はメインレースを強調表示
- 重賞がない日でも正常動作

---

## 設計思想

- 公式データをそのまま活かす構造
- 静的サイトとして最大限軽量化
- 自動更新によるメンテナンス不要設計
- 将来の開催構造変更にも対応可能
- 見た目はシンプル、内部は堅牢

---

## 技術構成

- Vanilla JavaScript
- DOM制御
- JRAカレンダーJSON解析
- GitHub Actions 自動取得
- GitHub Pages 配信

フレームワーク非依存設計。

---

## データフロー

GitHub Actions  
↓  
JRA公式カレンダーJSON取得  
↓  
data/jra/ に保存  
↓  
フロントエンドが読み込み表示
